# SPACE_IMAGES_FOR_TG

## tg_img_bot.py

Бот для отправки изображений и отложенных публикаций в телеграмм канал.

**Как установить:**

- Должен быть установлен `Python` версии 3.10
- Используйте `pip` для установки зависимостей

```bash
pip install -r requirements_bot.txt
```

**Запуск**

Получите токен для бота у [@Bot_Father](https://t.me/botfather) и id своего канала у [Get My Id](https://t.me/Getmyid_bot) (добавив его в группу). Создайте `.env` файл в корне проекта и скопируйте содержимое из `.env.example` в `.env.` Вставьте ваш реальный токен и id канала в файл `.env`. Также в `.env` файле можно задать время интервала между публикациями (в секундах). Используйте `--help` для подсказки. Режимы работы `--mode{img, auto} `, где *img* - публикация одного изображения, *auto* - автоматическая публикация фото из папки с временным интервалом. Задать интервал можно в файле `.env`. По умолчанию это 14400 секунд. Введите `--path` для указания пути к фото или к папке с изображениями.

**Пример ввода:**
- python .\tg_img_bot.py --mode img --path "E:\space_imgs\0_MessierCrater3d_vantuyne1024c.jpg.jpg"
- python .\tg_img_bot.py --mode auto --path "E:\Epic_photos"


## apod_nasa_imgs.py

Скрипт для скачивания фото дня или коллекции фото космоса от Nasa.

**Как установить:**

- Должен быть установлен `Python` версии 3+
- Клонируйте в корневую папку проекта `imgs_downloader.py`
- Используйте `pip` для установки зависимостей

```bash
pip install -r requirements.txt
```

**Запуск**

Получите токен на сайте [Nasa](https://api.nasa.gov/). Создайте `.env` файл в корне проекта и скопируйте содержимое из `.env.example в .env`. Вставьте ваш реальный токен в файл `.env`. Используйте `--help` для подсказки. Режимы скачивания `--mode{collection, day} `, где *collection* - коллекция из случайных фото, *day* - фото дня от Nasa.(не всегда есть фото дня от Nasa, в таком случае программа об этом сообщит). Укажите путь к папке или название для папки `--folder`. Также можно задать папку в файле `.env`. По умолчанию выставлен режим `{day}` и изображения сохраняться в папку *Images* в корень проекта. Скачивание может занять от 2 до 5 минут. Количество фото для коллекции настраивается в `.env` файле: `QUANTITY_APOD=5`. Используйте прокси, если получаете ошибку соединения.

**Пример ввода:**
- `python .\apod_nasa_imgs.py --mode collection`
- `python .\apod_nasa_imgs.py --mode day --folder apod`
- `python .\apod_nasa_imgs.py --mode day --folder E:\space_imgs`

- `python .\fetch_spacex_imgs.py --mode id --id 5eb87d46ffd86e000604b388`
- `python .\fetch_spacex_imgs.py --mode id --id 5eb87d46ffd86e000604b388 --folder spaceX`

## epic_photo_earth.py

Скрипт для скачивания фотографий земли из космоса от Nasa.

**Как установить:**

- Должен быть установлен `Python` версии 3+
- Клонируйте в корневую папку проекта `imgs_downloader.py`
- Используйте `pip` для установки зависимостей

```bash
pip install -r requirements.txt
```

**Запуск**

Получите токен на сайте [Nasa](https://api.nasa.gov/). Создайте `.env` файл в корне проекта и скопируйте содержимое из `.env.example в .env`. Вставьте ваш реальный токен в файл `.env`. Используйте `--help` для подсказки. Режимы скачивания `--mode{last, date} `, где *last* - самые свежие фото, *date* - фото с заданной даты. В моде `date` введите дату. Формат гггг-мм-дд. Пример (2026-04-26). Укажите путь к папке или название для папки `--folder`. Также можно задать папку в файле `.env`. По умолчанию выставлен режим `{last}` и изображения скачаются в папку *Images* в корень проекта. Скачивание может занять от 2 до 5 минут. Количество фото для коллекции настраивается в `.env` файле: `QUANTITY_EPIC=5`(Максимум в день Nasa выкладывает от 16 до 25 фото земли). Используйте прокси, если получаете ошибку соединения.

**Пример ввода:**

- python .\epic_photo_earth.py --mode date --date 2025-05-05
- python .\epic_photo_earth.py --mode date --date 2025-05-05 --folder E:\Epic_photos

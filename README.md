# SPACE_IMAGES_FOR_TG

## tg_img_bot.py

Бот для отправки изображений и отложенных публикаций в телеграмм канал.

**Как установить:**

- Должен быть установлен `Python` версии 3.10
- Используйте `pip` для установки зависимостей

```bash
pip install -r requirements_bot.txt
```

**Запуск**

Получите токен для бота у [@Bot_Father](https://t.me/botfather) и id своего канала у [Get My Id](https://t.me/Getmyid_bot) (добавив его в группу). Создайте `.env` файл в корне проекта и скопируйте содержимое из `.env.example` в `.env.` Вставьте ваш реальный токен и id канала в файл `.env`. Также в `.env` файле можно задать время интервала между публикациями (в секундах). Запустите программу и введите путь к изображению, которое надо опубликовать. Или нажмите Enter, укажите путь к папке с изображениями и бот будет публиковать картинки в случайном порядке с заданным интервалом в ваш телеграмм канал.

## apod_nasa_imgs.py

Скрипт для скачивания фото дня или коллекции фото космоса от Nasa.

**Как установить:**

- Должен быть установлен `Python` версии 3+
- Клонируйте в корневую папку проекта `imgs_downloader.py`
- Используйте `pip` для установки зависимостей

```bash
pip install -r requirements.txt
```

**Запуск**

Получите токен на сайте [Nasa](https://api.nasa.gov/). Создайте `.env` файл в корне проекта и скопируйте содержимое из `.env.example в .env`. Вставьте ваш реальный токен в файл `.env`. Запустите программу `apod_nasa_imgs.py`. Введите любой символ, чтобы скачать фото дня от Nasa, или нажмите Enter, чтобы скачать коллекцию фотографий. Скачивание может занять от 2 до 5 минут. После завершения программы в папке проекта появится папка *"Images"* со скачанными фотографиями. Если такая папка уже существует, фото загрузятся в эту папку. Используйте прокси, если получаете ошибку соединения.
